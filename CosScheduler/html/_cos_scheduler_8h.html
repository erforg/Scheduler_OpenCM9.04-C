Detailed Description

openCM / Arduino version of the 'co-operative', stackless tasking system (COS)

Project : co-operative Scheduler
Module : CosScheduler

A task-switch ist performed by means of deticated macros. These macros have to be called by the task-function directly. They must not be used in nested function calls.

void myTask2(CosTask_t *pt)
{   static int16_t count =0;
    COS_TASK_BEGIN(pt);

    SerialUSB.print("T2 started\r\n");
    while(count < 10)
    {    SerialUSB.print("T2 cnt=");
         SerialUSB.print(count++);
         SerialUSB.print("\r\n");
         COS_TASK_SLEEP(pt,_milliSecToTicks(500));
    }
    SerialUSB.print("T2 end\r\n");
    COS_TASK_END(pt);
}

Description
Priority base scheduler for Arduino and openCM. The task-list is sorted due to priority, scheduler will run the first task in the list that is in 'ready' state and has an expired 'sleep time'. The scheduler run in an endless loop.
