Node_t* _unlinkTaskFromTaskList ( Node_t *  root_pt,
CosTask_t *  task_pt 
)
Description
This routine searches the task-list for the node, pointing to the same task as task_pt. The node will be deleted from the list, its memory is freed. The task struct will not be deleted, it might be part of other lists (e.g semaphores have a list of waiting task). The root pointer of the task-list will change, if the node to be deleted is the first in the list. The function returns the (possibly updated) value of the root pointer to the task-list.
See also
_addTaskAtBeginningOfTaskList(), _newNode()
Parameters
root_pt- IN, pointer to first node in task-list
task_pt- IN, pointer to initialized task struct
Return values
pointerto first node in task-list
